# README


## Happyn2n

Happyn2n Windows Client 是针对N2N客户端包装的GUI程序，为N2N的用户提供简单易用的界面控制；

Happyn2n Windows Client 最初代码来源于(n2nedgegui项目)[https://sourceforge.net/projects/n2nedgegui/]，采用GPLV3授权，本项目同样采用GPLV3;
